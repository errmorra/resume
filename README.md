# Eric Morales — Hacker Portfolio

## 🚀 Deploy to GitHub Pages

1. **Create a new GitHub repo** (e.g. `portfolio` or `errmorra.github.io`)
2. **Upload** the `index.html` file to the root of the repo
3. Go to **Settings → Pages**
4. Set **Source** to `Deploy from a branch` → `main` → `/ (root)`
5. Hit **Save** — your site will be live at `https://yourusername.github.io/reponame`

> If you name the repo `errmorra.github.io`, it will deploy to `https://errmorra.github.io` directly.

## ✨ Features

### Recruiter / hiring-manager friendly
- 🛡 **Recruiter Mode toggle** — one click swaps the hacker theme for a clean, light, scan-friendly layout (preference saved in `localStorage`)
- ⬇️ **Download / print résumé** — generates a clean one-page résumé via the browser's Save-as-PDF dialog (dedicated print stylesheet)
- ✉️ **Copy email** button with confirmation toast
- 📊 **Impact-by-the-numbers** metrics with animated count-up (4+ yrs, 500+ staff trained, 35% phishing report lift, 5 tools, 110 NIST controls, 3 CompTIA certs)
- 🔎 **SEO + Open Graph + JSON-LD `Person` schema** so the site indexes well and previews cleanly when shared
- ♿ **Accessibility** — honors `prefers-reduced-motion`, keyboard-friendly, semantic structure

- ⚡ **Boot sequence** animation on load
- 🖋 **Typewriter** summary reveal

  - `whoami`, `skills`, `projects`, `grc`, `contact`, `certs`, `memberships`, `resume`, `theme`, `status`, `hack`, `clear`

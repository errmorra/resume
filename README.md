# Eric Morales — Hacker Portfolio

## 🚀 Deploy to GitHub Pages

1. **Create a new GitHub repo** (e.g. `portfolio` or `errmorra.github.io`)
2. **Upload** the `index.html` file to the root of the repo
3. Go to **Settings → Pages**
4. Set **Source** to `Deploy from a branch` → `main` → `/ (root)`
5. Hit **Save** — your site will be live at `https://yourusername.github.io/reponame`

> If you name the repo `errmorra.github.io`, it will deploy to `https://errmorra.github.io` directly.

## ✨ Features
- 🟢 **Matrix rain** background (katakana + hex chars)
- ⚡ **Boot sequence** animation on load
- 🖋 **Typewriter** summary reveal
- 💀 **Glitch** name effect
- 📊 **Animated skill bars** (scroll-triggered)
- 🔦 **Scanning line** on project cards
- 🖥 **Interactive terminal** — type `help` to explore commands:
  - `whoami`, `skills`, `projects`, `contact`, `certs`, `status`, `hack`, `clear`
- 🎯 **Custom cursor** with glow
- 📺 **CRT scanlines** overlay
- 🗂 **Sticky nav** with scroll tracking
